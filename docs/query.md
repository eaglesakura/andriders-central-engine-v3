# SQLite クエリメモ

## 統計を取得する

```
SELECT
	SESSION_ID,
	MIN(DATE) AS SORT_DATE,
	MAX(DATE),
	MAX(VALUE_HEARTRATE),
	MAX(VALUE_CADENCE),
	MAX(VALUE_SENSOR_SPEED),
	MAX(VALUE_GPS_SPEED),
	MAX(VALUE_FIT_CALORIES),
	MAX(VALUE_FIT_EXERCISE),
	MAX(VALUE_RECORD_DISTANCE_KM),
	MAX(VALUE_RECORD_SUM_ALT_METER),
	MAX(VALUE_ACTIVE_DISTANCE_KM),
	MAX(VALUE_ACTIVE_TIME_MS)
FROM DB_SESSION_POINT
GROUP BY SESSION_ID
ORDER BY SORT_DATE ASC
```
